Technologies for Signal Separation and Enhancement

We work on a range of signal processing technologies such as:

Blind Source Separation

Blind source separation technic separates the signals from their mixtures without any information about the original signals or the mixing process. The method is suitable for the separation of speech, biomedical or communication signals from their mixtures.

Beamforming

The technic can pick up the desired signal from a spatially separated multisource environment. The current technology will allow the user to select the desired source through video based touch interface.

Sound classification

The technology can identify different sound signals. The technic is useful for event detection in security application.

Source localization and direction of arrival estimation

We have developed technics for source localization and direction of arrival estimation for both single and multiple sources.
